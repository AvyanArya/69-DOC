Connect your data

Because this file was opened on its own, the browser keeps its storage separate from Lumera’s — so there’s nothing to read yet. Pick the easiest option:

Option A — instant, zero setup (recommended)

You don’t need this file at all. The same comprehensive live dashboard is built into Lumera: open Lumera.html → Settings is hidden; just go to the address bar and add #/analytics (or paste lumera.html#/analytics), enter the key, and it shows your real data live.

Option B — load a snapshot file

In Lumera’s analytics click Export, then load that file here:

Option C — live, shared (one command)

Put Lumera.html and admin-portal.html in one folder, open a terminal there and run:

python3 -m http.server 8000

Then open http://localhost:8000/Lumera.html and http://localhost:8000/admin-portal.html. Now they share an origin and this portal updates live automatically.
